# Shamba Palace Farm Machinery Booking System.

## Project Overview
Shamba Palace is a web-based system for managing agricultural machinery rentals. 
The system allows users to book machinery and administrators to manage bookings, machinery inventory, and user accounts.

## Installation and Setup

### Using XAMPP
1. Download and install XAMPP from https://www.apachefriends.org/
2. Copy the project folder (shamba) to `C:\xampp\htdocs\shamba` (Windows) or `/opt/lampp/htdocs/shamba` (Linux/Mac)
3. Start XAMPP Control Panel and start Apache and MySQL services
4. Open phpMyAdmin at http://localhost/phpmyadmin
5. Create a new database named `shamba_palace`
6. Import the database schema from `database/shamba.sql`
7. Access the application at http://localhost/shamba

### Using LAMPP (Linux)
1. Install LAMPP: `sudo apt-get install lampp`
2. Copy project folder (shamba) to `/opt/lampp/htdocs/shamba`
3. Start LAMPP services: `sudo /opt/lampp/lampp start`
4. Open phpMyAdmin at http://localhost/phpmyadmin
5. Create database named `shamba_palace`
6. Import database schema from `database/shamba.sql`
7. Access the application at http://localhost/shamba

## User Types:
1. Regular Users
   - Can register new accounts
   - Book available machinery
   - View their booking history
   - Reset their password
   - View machinery status and pricing

2. Administrators:
   - Manage machinery inventory (add, edit, delete)
   - Approve or reject booking requests
   - View all bookings
   - Manage user accounts
   - Update system settings
   - Reset admin password

## Login Credentials
1. User Login:
   - Access via login.html
   - Required fields: username, password
   - Registration available for new users

2. Admin Login:
   - Access via login.html (admin toggle)
   - Required fields: username, password
   - Default admin email: admin@shambapalace.com

## Workflow
1. User Booking Process:
   - User logs in
   - Views available machinery
   - Selects machinery and enters booking details
   - Submits booking request
   - Receives status updates

2. Admin Management Process:
   - Admin logs in
   - Reviews pending bookings
   - Approves or rejects with reason
   - Manages machinery inventory
   - Updates system settings
